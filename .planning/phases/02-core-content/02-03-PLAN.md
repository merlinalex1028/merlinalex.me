<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| reader→Twikoo | Comment content submitted by untrusted readers |
| Twikoo→page | Comment HTML rendered on article pages (untrusted) |
| CDN→page | Twikoo JS loaded from jsdelivr CDN |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-06 | Tampering | XSS via Twikoo comments | mitigate | Twikoo sanitizes all comment input internally; CSP headers from Cloudflare Pages provide defense-in-depth |
| T-02-07 | Tampering | Twikoo CDN script tampering | accept | jsdelivr serves integrity-hashed bundles; pinned version (`@1.7.11`) prevents auto-updates |
| T-02-08 | Information Disclosure | envId exposure | accept | envId is a public Vercel URL, not a secret; it identifies the backend instance but grants no privileged access |
| T-02-SC | Tampering | CDN script load | accept | Twikoo loaded from jsdelivr CDN (well-known, integrity-checked); not an npm install |
</threat_model>
