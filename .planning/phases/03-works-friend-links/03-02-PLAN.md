Create the multi-image lightbox overlay for the Creations gallery. Extends the medium-zoom pattern from Phase 2 into a full gallery lightbox. Per D-03 (locked decision):

**src/components/works/CreationLightbox.astro** — Client-side lightbox component using `client:idle` directive. Implementation:

- HTML structure: A `position: fixed` overlay covering the full viewport with `z-index: 100` (above Header's 50), `background: rgba(0, 0, 0, 0.9)`. Contains:
  - Close button (top-right): `✕` with `aria-label="关闭"`
  - Previous/Next buttons (left/right sides): `‹` / `›` with `aria-label="上一张"` / `aria-label="下一张"`
  - Image container (centered): `<img>` with `max-width: 90vw; max-height: 85vh; object-fit: contain;`
  - Info panel (bottom): Title (`<h3>`) and description (`<p>`) with `color: white`
  - Counter: `"2 / 5"` style indicator showing current position

- Client script logic:
  - Listen for clicks on `.creation-card` elements (event delegation on document)
  - On click: read `data-images` (JSON array), `data-title`, `data-description`, `data-index` from the clicked card
  - Parse `data-images` as JSON to get image URL array
  - Show overlay, set first image, update title/description
  - Track `currentIndex` state; update image src on prev/next
  - Close: hide overlay, reset state
  - Keyboard: `document.addEventListener('keydown', ...)` — Escape closes, ArrowLeft prev, ArrowRight next. Only active when lightbox is open.
  - Click-outside: clicking the overlay background (not the image or buttons) closes lightbox
  - Prev/Next buttons hidden when array has only 1 image
  - Re-initialize after View Transitions: `document.addEventListener('astro:after-swap', reinit)`

- CSS (scoped via `<style>`):
  - Overlay: `position: fixed; inset: 0; z-index: 100; display: none;` (shown via `.active` class: `display: flex`)
  - Image: `max-width: 90vw; max-height: 85vh; object-fit: contain; border-radius: 8px;`
  - Buttons: `background: rgba(255,255,255,0.1); color: white; border: none; cursor: pointer; border-radius: 50%; width: 48px; height: 48px; font-size: 24px;` Hover: `background: rgba(255,255,255,0.2)`
  - Transition: `opacity 200ms` on overlay appear/disappear
