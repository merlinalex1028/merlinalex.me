    - Import `PagefindConfig` from `astro-pagefind/components/PagefindConfig.astro`
    - Render `<PagefindConfig />` to initialize Pagefind
    - Render `<pagefind-searchbox>` web component with:
      - `placeholder="搜索文章..."`
      - `shortcut="/"` (press / to focus search)
      - `max-results="10"`
    - Style the searchbox to match the site theme: use CSS variables for colors, border-radius, font
    - Add custom CSS for the search results dropdown: `--pagefind-ui-primary: var(--color-accent)`, `--pagefind-ui-text: var(--color-fg)`, `--pagefind-ui-background: var(--color-surface)`, `--pagefind-ui-border: var(--color-border)`

    **4. Update `src/components/core/Header.astro`:**
    - Import `SearchBar` from `../search/SearchBar.astro`
    - Add `<SearchBar />` inside `header-actions` div, before ThemeSwitcher
    - Style: search input should be compact in the header (icon + expandable input)

    **5. Create `src/components/archive/TagCloud.astro`:**
    - Props: `tagCounts` (Record<string, number>) — tag name to count mapping
    - Filter out tags with count <= 1 (hide 0 and 1-post tags per D-06)
    - Compute font size: map count to 14px-28px range using percentile scaling:
      - Find min and max counts from filtered tags
      - For each tag: `fontSize = 14 + ((count - min) / (max - min)) * 14` px
    - Render tags as links: `<a href="/articles?tag={tag}">` with computed font-size
    - Use CSS variables: `--color-accent`, `--color-fg-muted`
    - Tags wrap naturally with `flex-wrap: wrap` and `gap: 8px`

    **6. Create `src/components/archive/ArchiveList.astro`:**
    - Props: `articles` (sorted by publishedAt descending)
    - Group articles by year (extract year from publishedAt)
    - Render year headers (e.g., "2026", "2025") as section headings
    - Under each year, list articles as simple rows: date (MM-DD) + title (linked to article)
    - Use CSS: clean list with subtle year dividers

    **7. Create `src/pages/archive.astro`:**
    - Import `getCollection('articles')` to get all non-draft articles
    - Import `TagCloud` and `ArchiveList`
    - Compute tagCounts: iterate articles, extract tags, count occurrences
    - Sort articles by publishedAt descending for ArchiveList
    - Render page with TagCloud at top, ArchiveList below
    - Use `BaseLayout` with title "归档 — merlinalex.me"
